## ✨ Features

- 🔢 **CIDR calculator** — enter any IP range and get network address, broadcast, host range instantly
- 📡 **Subnet splitter** — divide a network block into equal or variable subnets automatically
- 🗺️ **Visual subnet map** — see your address space laid out clearly
- 🔄 **IPv4 and IPv6 support** — works for both address families
- 📋 **Copy-ready output** — all values formatted for firewall rules and cloud VPC configs
- 🧮 **VLSM calculator** — variable-length subnet masking for complex network designs
- ⚡ **Zero server calls** — all logic runs client-side, works offline too

## 🚀 Quick Start

```bash
git clone https://github.com/jayblast-spec/subnetpilot.git
cd subnetpilot
npm install
npm run dev
```

Or just use the [live tool](https://subnetpilot.vercel.app) — no install needed.
